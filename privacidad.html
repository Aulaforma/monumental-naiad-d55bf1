2. Finalidad de esta política

Esta política informa al usuario qué datos recopila Aula Forma, para qué los utiliza, cómo los protege, con qué servicios externos puede operar y cómo el usuario puede ejercer sus derechos.

Al registrarse o utilizar la plataforma, el usuario acepta esta Política de privacidad.

3. Datos que Aula Forma puede recopilar

Aula Forma podrá recopilar los siguientes datos:

a) Datos de registro

  • Nombre completo.
  • Correo electrónico.
  • Contraseña cifrada o protegida.
  • Tipo de usuario.
  • Fecha de registro.
  • Estado de la cuenta.

b) Datos de uso

  • Créditos disponibles.
  • Créditos utilizados.
  • Historial de documentos generados.
  • Asignatura seleccionada.
  • Curso o nivel indicado por el usuario.
  • Tipo de instrumento solicitado.
  • Fecha y hora de generación.

c) Datos técnicos

  • Dirección IP.
  • Navegador utilizado.
  • Dispositivo.
  • Registros técnicos de acceso.
  • Información necesaria para seguridad, funcionamiento y prevención de abuso.

d) Material ingresado por el usuario

El usuario podrá ingresar textos, instrucciones, contenidos pedagógicos, objetivos, fragmentos de clases, indicaciones o material base para generar documentos.

Aula Forma utilizará dicho material únicamente para prestar el servicio solicitado.

4. Finalidades del tratamiento

Los datos se utilizarán para:

  • Crear y administrar cuentas de usuario.
  • Permitir inicio de sesión.
  • Entregar créditos gratuitos o comprados.
  • Generar documentos evaluativos en Word.
  • Registrar historial de uso.
  • Gestionar soporte técnico.
  • Procesar pagos, cuando corresponda.
  • Prevenir fraudes, abusos o usos indebidos.
  • Mejorar el funcionamiento de la plataforma.
  • Cumplir obligaciones legales o administrativas.

5. Datos sensibles y datos de estudiantes

Aula Forma no solicita ni requiere datos sensibles de estudiantes.

El usuario no debe ingresar:

  • RUT de estudiantes.
  • Diagnósticos médicos.
  • Informes psicológicos.
  • Datos de salud.
  • Situaciones familiares privadas.
  • Domicilios.
  • Antecedentes personales sensibles.
  • Información que identifique innecesariamente a niños, niñas o adolescentes.

Si el usuario requiere trabajar con casos o ejemplos, deberá anonimizar la información antes de ingresarla.

Aula Forma no se hace responsable por la carga indebida de datos sensibles realizada por el usuario.

6. Uso de servicios externos

Para operar correctamente, Aula Forma podrá utilizar servicios externos, tales como:

  • Servicios de alojamiento web.
  • Servicios de autenticación.
  • Bases de datos.
  • Plataformas de pago.
  • Servicios de inteligencia artificial.
  • Herramientas de generación o almacenamiento de documentos.

Entre estos servicios podrían encontrarse Netlify, Supabase, OpenAI, Flow, Mercado Pago u otros proveedores técnicos.

Estos servicios podrán procesar datos estrictamente necesarios para el funcionamiento de la plataforma.

7. Uso de inteligencia artificial

Aula Forma utiliza inteligencia artificial para generar documentos a partir del material ingresado por el usuario.

El usuario entiende que parte del contenido entregado podrá ser procesado por servicios de inteligencia artificial con el único objetivo de generar el documento solicitado.

El usuario no debe ingresar información confidencial, sensible o privada que no sea necesaria para la generación del instrumento.

8. Conservación de datos

Aula Forma conservará los datos mientras la cuenta del usuario se mantenga activa o mientras sea necesario para prestar el servicio, cumplir obligaciones legales, resolver reclamos, mantener registros de compra o prevenir usos indebidos.

El usuario podrá solicitar la eliminación de su cuenta y datos personales escribiendo a aulaforma2026@gmail.com.

La eliminación podrá estar sujeta a conservación mínima de ciertos registros cuando exista obligación legal, tributaria, contable o de seguridad.

9. Seguridad de la información

Aula Forma adoptará medidas razonables para proteger los datos personales y evitar acceso no autorizado, pérdida, alteración o uso indebido.

Entre las medidas se consideran:

  • Uso de autenticación.
  • Protección de contraseñas.
  • Acceso restringido a datos.
  • Protección de rutas privadas.
  • Uso de funciones seguras para la conexión con servicios externos.
  • No exposición de claves API en el frontend.

Sin embargo, ningún sistema digital puede garantizar seguridad absoluta.

10. Derechos del usuario

El usuario podrá solicitar:

  • Acceso a sus datos personales.
  • Rectificación de datos incorrectos.
  • Elimación de su cuenta.
  • Información sobre el uso de sus datos.
  • Oposición o limitación al tratamiento, cuando corresponda.

Para ejercer estos derechos, deberá escribir a: aulaforma2026@gmail.com

La solicitud deberá indicar nombre, correo registrado y requerimiento específico.

11. Comunicaciones

Aula Forma podrá enviar comunicaciones al correo del usuario para:

  • Confirmar registro.
  • Informar compras o créditos.
  • Responder solicitudes de soporte.
  • Notificar cambios importantes en el servicio.
  • Enviar información funcional relacionada con la cuenta.

El usuario podrá solicitar dejar de recibir comunicaciones promocionales, si existieran.

12. Menores de edad

Aula Forma está dirigida principalmente a docentes, profesionales de la educación, equipos técnicos y usuarios adultos.

Si un menor de edad utiliza la plataforma, deberá contar con autorización de su padre, madre, tutor o representante legal.

13. Cambios en esta política

Aula Forma podrá modificar esta Política de privacidad para ajustarla a cambios legales, técnicos o funcionales.

La versión actualizada estará disponible en la página web.
